# VeinBrains dApp Frontend v0.1

## Getting Started
- BUIDL.

### Install / Contribute

Clone repo and access the project directory.

```bash
cd dapp-landingpage
```

Install dependencies.

```bash
yarn install
```

Serve with hot reload at <http://localhost:1337>.

```bash
yarn run dev
```

### Lint

```bash
yarn run lint
```

### Typecheck

```bash
yarn run typecheck
```

### Build

```bash
yarn run build
```

### Test

```bash
yarn run test
```
